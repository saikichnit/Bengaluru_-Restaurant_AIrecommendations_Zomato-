# Phase 5: LLM Recommendation Engine

## Responsibility
Generate personalized, human-friendly recommendations using an LLM.

## Key Tasks
- **Prompt Engineering**: Develop templates that include user context and restaurant candidates.
- **LLM Integration**: Implement the client specifically for the **Groq LLM API** for high-speed inference.
- **Structured Output**: Ensure the LLM follows a strict JSON schema for restaurant recommendations.
- **Prompt Engineering**: Develop templates optimized for Groq (e.g., Llama-3-70b/8b).
- **Safety**: Implement guardrails to prevent hallucination of restaurant features.
